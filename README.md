# Korea Pinoy Media Academy (KPMA) - Landing Page

The official landing page for the **Korea Pinoy Media Academy (KPMA)**, a professional creative microcredential bootcamp bridging Korean production excellence with Filipino creativity.

🌐 **Live:** [academy.pinoyseoul.com](https://academy.pinoyseoul.com)

---

## About KPMA

KPMA offers intensive, portfolio-building programs (1-2 weeks) in specific creative skills targeting students, recent graduates, and young professionals in the Philippines. Graduates receive a PS Media Professional Microcredential Certificate and access to the Seoul Talents Network.

### Bootcamp Tracks

| Track | Target Students | Duration |
|-------|-----------------|----------|
| K-Brand Strategy | Marketing, Advertising, Business | 2 Weekends (32 hrs) |
| K-Creative Production | Film, Broadcasting, Multimedia Arts | 1 Week (40 hrs) |
| Visual Korea Design Sprint | Graphic Design, Visual Communication | 1 Week (35 hrs) |
| Social Media Accelerator | Digital Media, Communication | 2 Weekends (32 hrs) |
| K-Performance Intensive | Performing Arts, Theater, Music | 1 Week (40 hrs) |
| Entertainment Reporting | Journalism, Broadcasting | 2 Weekends (32 hrs) |

## Tech Stack

- **HTML5** + **CSS3** + **Vanilla JavaScript**
- **GitHub Pages** for hosting
- **Cloudflare** for DNS and CDN
- **Google Search Console** for SEO monitoring

## Deployment

This site is automatically deployed via GitHub Pages on push to `main`. The custom domain `academy.pinoyseoul.com` is configured via the CNAME file and Cloudflare DNS.

## SEO Features

- Structured Data (JSON-LD): EducationalOrganization, Course, FAQ, Breadcrumb
- Open Graph and Twitter Card meta tags
- XML Sitemap (auto-generated via GitHub Action)
- robots.txt
- Semantic HTML5 with proper heading hierarchy
- Optimized for Core Web Vitals
